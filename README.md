# Android Assignment 🔗
# Features Implemented:
1. App can handle integer as well as decimal value.
2. Handle edge cases to make the app more robust.
3. Trailing Zero Removal in Decimal Numbers i.e '12.0' will be displayed as 12 but '12.01' will remain unchanged and be displayed as '12.01' to inhance user experience.
4. The input field is cleared when displaying the final result in Activity_01, allowing the user to enter another number more conveniently.
5. Simple UI.
